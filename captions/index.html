Captions

The captions are automatically created using the alt attribute of images. Moreover, if it doesn't exist, it is created using the title attribute of the links.

For example, the following entry have "I'm only in the alt" as caption. {% highlight html %} {% endhighlight %}

The following entry hasn't an alt, but it has a title. In this case the plugin will set "I'm only in the title" as caption. {% highlight html %} {% endhighlight %}

In this last example, we have both the title and the alt. In this case the plugin brings the text from the alt, and the caption will be "I'm in the alt". {% highlight html %} {% endhighlight %}

The plugin creates the captions inserting, for each entry, an extra div element just after the img element. The latter is build to have the class caption. To have custom captions you need only to insert this element on your own, with the content that you want. The plugin will skip the creation of the captions for the entries that already have a caption in the entry. {% highlight html %}
